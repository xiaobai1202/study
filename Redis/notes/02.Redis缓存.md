# Redis 缓存

---

缓存就是数据交换的缓冲区，称作cache，是贮存数据的临时地方，一般读写性能较高。

缓存的作用：

1. 降低后端负载
2. 提高读写效率，降低响应时间

使用缓存的成本

1. 数据一致性成本
2. 代码维护成本
3. 运维成本
